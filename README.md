## Codepath Intro to Mobile App Development IOS Prework

### App Description
IOS app in which a button can be pressed to change string color, as well as a button can be pressed to change background color and a third button can be pressed to change what the string in the label says according to input given by user in a text field. If text field is empty and change string button is pressed then a default "Goodbye ðŸ‘‹" string appears on the label. if string, string color and background color are changed and the screen is tapped on the side the default values that appear on app startup will be reset to.
